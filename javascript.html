  • About this course
    about

    This course is about building websites and applications using javascript as basics for frontend and backend programming and some of the most important related technologies, such as html, css and NodeJs.

    We have made this path by pulling the best resources from around the web and combining them in a unique way, to help ease this incredible journey for you.

    JavaScript is one of the most used programming langauges in the world today, and will continue to dominate the field of web programming for at least a couple of years, so, learn js :)

    prerequisites

    You need to know English :D, have a basic knowledge of computers, how to type et cetera. No big deal, if you know some of the things below feel free to skip them.

    what should i know after this

    After completing most of the stuff that is in our curriculum and of course, doing the coding part yourself, you will be able to build static and dynamic websites, to build stuff like login systems, simple social media sites, basics of command line use and so on. You will do that by using JavaScript and its frameworks and libraries.

    how much time to complete

    Obviously it depends on how much time you spend each day blabla,but rougly speaking, if you spend about 3.5 hours each day on learning and coding, it shouldn't take much more than 10-11 months to complete most of this course
